# Logist
Программа для оптимизации логистического процесса с закрытым исходным кодом. В репозитории находятся: исполнительный файл (Логист.exe) и папка Resource с файлами Distances.csv (таблица расстояний между городами, граф), Towns.csv (массив городов и сведений о них), Users.txt (база учётных записей)
